# Contributing

Thanks for helping improve Z3 SMT Game.

## Local Setup

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `make install-hooks`.
4. Use Conventional Commits for commit messages.

## Quality Bar

- Keep user-facing flows keyboard accessible.
- Add focused tests for solver logic and UI behavior.
- Do not commit secrets, `.env` files, generated logs, or local machine paths.
